XgUIListBox

XgUIListBox Class

new XgUIListBox(selector, option) → {object}

새로운 리스트 박스 인스턴스를 생성한다.

Parameters:
Name Type Argument Description
selector string

CSS 셀렉터
option object <optional>

리스트 박스 옵션
Options:
Name Type Default Description
xgInitOption object {} 데이터 셋 및 데이터 테이블 정보를 설정한다. 자세한 정보는 여기를 참조한다.
xgBindOption object {} 데이터 바인딩 정보를 설정한다. 자세한 정보는 여기를 참조한다.
disabled boolean false 비활성화 여부를 설정한다.
width number｜string null 폭을 설정한다.
height number｜string null 높이를 설정한다.
theme string '' 테마를 설정한다. 테마설정
multiple boolean false 다중 선택 기능 활성화 여부를 설정한다.
multipleextended boolean false 확장된 다중 선택 기능 활성화 여부를 설정한다. 키보드의 키( ctrl[ 추가 ], shift[ 범위 선택 ] ) 눌림과 마우스 왼쪽 클릭을 이용한다.
selectedIndex number -1 특정 위치의 아이템을 선택된 상태로 설정한다.
source array null 아이템 요소들이 담긴 참조 데이터를 설정한다.
displayMember string '' 'source' 배열안의 아이템이 오브젝트 일 경우 팝업 아이템 화면상 노출할 속성을 설정한다.
valueMember string '' 'source' 배열안의 아이템이 오브젝트 일 경우 팝업 아이템을 선택 시 값으로 취급할 속성을 설정한다.
scrollBarSize number 17 스크롤 바의 크기를 설정한다.
enableHover boolean true 아이템 요소에 마우스를 올렸을 때 구분을 위한 Hover CSS 사용여부를 설정한다.
searchMode string 'startswith' 사용자가 키를 입력 시 ( 리스트 박스에 포커스가 지정되어 있을 때 ) 소스 데이터와 어떤식으로 비교할지를 설정한다.
  • 'none' : 비교하지 않음
  • 'contains' : 포함( 대소문자 구분 )
  • 'containsignorecase' : 포함
  • 'equals' : 같음( 대소문자 구분 )
  • 'equalsignorecase' : 같음
  • 'startswith' : ~으로 시작( 대소문자 구분 )
  • 'startswithignorecase' : ~으로 시작
  • 'endswith' : ~으로 끝남( 대소문자 구분 )
  • 'endswithignorecase' : ~으로 끝남
enableSelection boolean true 리스트 박스의 아이템의 선택가능 여부를 설정한다.
itemHeight number -1 아이템 요소의 높이를 설정한다.
checkboxes boolean false 리스트 아이템 요소 옆에 체크박스 기능을 활성화 시킬지 여부를 설정한다.
filterable boolean false 필터링 기능 활성화 여부를 설정한다.
filterHeight number 27 필터 값 입력 컨트롤의 높이를 설정한다.
filterDelay number 100 필터 값 입력 후 필터링 결과물을 표시하기 까지의 지연시간을 설정한다.
filterPlaceHolder string 'Lookin for' 필터 값이 비어있을 때 표시할 문자열을 설정한다.
autoHeight boolean false 아이템 요소의 갯수에 따라 리스트 박스의 높이를 자동으로 지정할지 여부를 설정한다.
hasThreeStates boolean true 3가지 상태를 지원할지 여부를 설정한다.
( null 일때 [ 값이 없을 때 ], true 일때, false 일때 )
allowDrop boolean false 드래그 앤 드랍으로 아이템 요소의 위치를 변경할 수 있는 기능의 활성화 여부를 설정한다.
dropAction string 'default' 드랍된 아이템 요소를 어떻게 반영할지 설정한다.
  • 'default' : 선택된 아이템을 드랍된 위치로 이동한다.
  • 'copy' : 드랍된 위치에 선택된 아이템의 복사본을 추가한다.
  • 'none' : 드랍되더라도 위치 변경 또는 추가를 하지않는다.
dragStart function null 아이템 요소가 드래그 될 때 수행할 콜백함수를 설정한다. 해당 함수는 boolean 타입을 반환하여야 하며 반환값에 따라 해당 아이템 요소의 드래깅 여부가 결정된다.
전달 인자 : item
dragEnd function null 아이템 요소가 드래그 될 때 수행할 콜백함수를 설정한다. 해당 함수는 boolean 타입을 반환하여야 하며 반환값에 따라 드랍된 요소를 'dropAction' 설정에 따라 동작 여부가 결정된다.
전달 인자 : dragItem, dropItem
  • dragItem : 드래그 중인 아이템 요소
  • dropItem : 드랍되는 위치의 아이템 요소
renderer function null 아이템 요소를 어떻게 표현할지 설정한다. 해당 함수의 반환값이 렌더링 된다.
전달 인자 : index, label, value

xgInitOption

Name Type Default Description
xgDataSet object|string null 데이터 셋을 설정한다. 데이터 셋 오브젝트 또는 데이터 셋 아이디를 지정할 수 있다.
xgDataTable object|string null 데이터 테이블을 설정한다. 데이터 테이블 오브젝트 또는 데이터 테이블 아이디를 지정할 수 있다.
해당 데이터 테이블은 같이 지정된 데이터 셋에 추가된 상태여야 한다.
xgBindOption

Name Type Default Description
mode string 'read' 데이터 바인딩 모드를 설정한다. 'bind.type' 이 'row' 일 경우에는 'read' 모드만 지원된다.
[ 'read', 'write' ] 중 택일
  • 'read' : 해당 컴퍼넌트의 변경사항이 타 컴퍼넌트에게 전달되지 않는다.
  • 'write' : 해당 컴퍼넌트의 변경사항이 타 컴퍼넌트에게 전달된다.
sync boolean true 데이터 테이블 로우의 상태값 및 제약조건의 반영 여부를 설정한다.
  • true : 로우의 상태값 및 제약조건과 상호작용( 자동 비활성화 ) 기능을 사용한다. 선택된 로우가 삭제될 경우 수정불가능 상태로 변경된다. 또한 서버와 동기화 된 'key', 'sequence' 컬럼을 바라보는 경우 수정불가능 상태로 변경된다.
  • false : 로우의 상태값 및 제약조건과 상호작용( 자동 비활성화 ) 기능을 사용하지 않는다.
bind object {} 오브젝트 구조는 여기를 참조한다.
bind object

Name Type Default Description
type string 'column' 데이터 테이블의 바인딩 목표를 설정한다. 데이터 테이블의 현재 선택된 로우를 기준으로 동작된다.
'column' 모드만 지원된다.
column string|number null 'column' type으로 설정되었을 때 바인딩 할 컬럼을 지정한다. 컬럼의 이름 또는 컬럼의 인덱스로 지정할 수 있다. ( 데이터 테이블 기준 )
item object in source array

요소에 추가된 아이템은 하기의 속성을 지닌 오브젝트로 변환되어 지정된다.

Name Description
label 리스트에 보여질 'displayMember'
value 실제 값으로 취급될 'valueMember'
disabled 아이템의 비활성화 여부
checked 아이템의 체크상태 여부
hasThreeStates 아이템의 체크상태를 3단계로 구분지어진지 여부
html 아이템의 보여지는 마크업
group 아이템의 그룹정보
Listens to Events:
  • select 아이템 요소가 선택될 때 발생한다
  • unselect 아이템 요소가 선택해제 될 때 발생한다
  • change 선택된 아이템 요소가 변경될 때 발생한다
  • checkChange 아이템 요소의 체크 상태가 변경될 때 발생한다
  • dragStart 아이템 요소가 드래그 될 때 발생한다
  • dragEnd 아이템 요삭 드랍 될 때 발생한다
  • bindingComplete 참조 아이템 배열이 로드가 된 후 발생한다
Returns:

$xgListBox

Type
object
Examples
var listBoxOption = {
    xgInitOption: {
        xgDataSet: dataSet,
        xgDataTable: 'dataTableName'
    },
    xgBindOption: {
        mode: 'write',
        bind: {
            type: 'column',
            column: 'columnName'
        }
    },
    width: 300,
    height: 300,
    ...
    source: [ { text: 'foo', value: 0 }, { text: 'bar', value: 1 } ]
    displayMember: 'text'
    valueMember: 'value'
};

// module 방식
var $xgListBoxModule = new XgUIListBox( '#xg-list-box', listBoxOption );

// plugin 방식
var $xgListBoxPlugin = $( '#xg-list-box' ).xgListBox( listBoxOption );
