XgUIInput

XgUIInput Class

new XgUIInput(selector, option) → {object}

새로운 입력 컨트롤 인스턴스를 생성한다.

Parameters:
Name Type Argument Description
selector string

CSS 셀렉터
option object <optional>

입력 컨트롤 옵션
Options:
Name Type Default Description
xgInitOption object {} 데이터 셋 및 데이터 테이블 정보를 설정한다. 자세한 정보는 여기를 참조한다.
xgBindOption object {} 데이터 바인딩 정보를 설정한다. 자세한 정보는 여기를 참조한다.
disabled boolean false 비활성화 여부를 설정한다.
width number｜string null 폭을 설정한다.
height number｜string null 높이를 설정한다.
opened boolean false 추천 단어 팝업을 열려있는 상태로 지정할지 여부를 설정한다. (초기화시 동작 문제 발견)
theme string '' 테마를 설정한다. 테마설정
searchMode string 'default' 사용자가 값을 입력할 때 'source'의 내용과 매칭할 방법을 설정한다.
  • 'none' : 매칭을 시도하지 않는다.
  • 'contains' : 포함( 대소문자 구분 )
  • 'containsignorecase' : 포함
  • 'equals' : 같음( 대소문자 구분 )
  • 'equalsignorecase' : 같음
  • 'startswith' : ~ 으로 시작( 대소문자 구분 )
  • 'startswithignorecase' : ~으로 시작
  • 'endswith' : ~으로 끝남( 대소문자 구분 )
  • 'endswithignorecase' : ~으로 끝남
source array｜function [] 추천 단어를 제공하기 위한 참조 데이터를 설정한다. 함수로 설정 시 2개의 인자가 해당 함수의 인자로 전달된다.
전달 인자 : inputValue, callbackFunction
items number 8 팝업에 표시할 최대 아이템 수를 설정한다.
minLength number 1 추천 단어를 매칭하기 위한 입력값의 최소 길이를 설정한다.
placeHolder string '' 입력값이 없을 때 보여질 문자열을 설정한다. IE9 이하에서는 해당 옵션이 지원되지 않는다.
popupZIndex number 20000 팝업의 z-index 값을 설정한다.
displayMember string '' 'source' 배열안의 아이템이 오브젝트 일 경우 팝업 아이템 화면상 노출할 속성을 설정한다.
valueMember string '' 'source' 배열안의 아이템이 오브젝트 일 경우 팝업 아이템을 선택 시 값으로 취급할 속성을 설정한다.
query string '' 쿼리를 설정한다.
renderer function null 추천 단어를 선택 후 입력 컨트롤의 값을 갱신할 함수를 설정한다. 함수의 반환 값이 입력 컨트롤의 값에 설정된다.
전달 인자 : itemValue, inputValue

xgInitOption

Name Type Default Description
xgDataSet object｜string null 데이터 셋을 설정한다. 데이터 셋 오브젝트 또는 데이터 셋 아이디를 지정할 수 있다.
xgDataTable object｜string null 데이터 테이블을 설정한다. 데이터 테이블 오브젝트 또는 데이터 테이블 아이디를 지정할 수 있다.
해당 데이터 테이블은 같이 지정된 데이터 셋에 추가된 상태여야 한다.
xgBindOption
Name Type Default Description
mode string 'read' 데이터 바인딩 모드를 설정한다. 'bind.type' 이 'row' 일 경우에는 'read' 모드만 지원된다.
[ 'read', 'write' ] 중 택일
  • 'read' : 해당 컴퍼넌트의 변경사항이 타 컴퍼넌트에게 전달되지 않는다.
  • 'write' : 해당 컴퍼넌트의 변경사항이 타 컴퍼넌트에게 전달된다.
sync boolean true 데이터 테이블 로우의 상태값 및 제약조건의 반영 여부를 설정한다.
  • true : 로우의 상태값 및 제약조건과 상호작용(자동 비활성화) 기능을 사용한다. 선택된 로우가 삭제될 경우 수정불가능 상태로 변경된다. 또한 서버와 동기화 된 'key', 'sequence' 컬럼을 바라보는 경우 수정불가능 상태로 변경된다.
  • false : 로우의 상태값 및 제약조건과 상호작용(자동 비활성화) 기능을 사용하지 않는다.
immediately boolean false 'write' 모드 일 때 변경사항을 즉시 전파할지 여부를 설정한다. false로 설정할 경우 컴퍼넌트에서 포커스를 잃을 때 변경사항을 전파한다.
bind object {} 오브젝트 구조는 여기를 참조한다.
bind object
Name Type Default Description
type string 'row' 데이터 테이블의 바인딩 목표를 설정한다. 데이터 테이블의 현재 선택된 로우를 기준으로 동작된다.
['row', 'column'] 중 택일
column string|number null 'row' type으로 설정되었을 때 바인딩 할 컬럼을 지정한다. 컬럼의 이름 또는 컬럼의 인덱스로 지정할 수 있다. (데이터 테이블 기준)
Listens to Events:
  • change - 입력 값이 변경된 후 포커스를 잃을 때 발생한다
  • select - 추천 단어 팝업의 아이템을 선택할 때 발생한다
  • open - 추천 단어 팝업이 열릴 때 발생한다
  • close - 추천 단어 팝업이 닫힐 때 발생한다
Returns:

$xgInput

Type
object
Examples
var source = ['foo', 'bar', 'miz', 'roo', 'key'];

// module 방식
var $xgInputModule = new XgUIInput( '#xg-input', {
    xgInitOption: {
        xgDataSet: dataSet,
        xgDataTable: dataTable
    },
    xgBindOption: {
        mode: 'write',
        immediately: true,
        bind: {
            type: 'column',
            column: 'columnName'
        }
    },
    width: '30%',
    height: 40,
    searchMode: 'equal',
    placeHolder: 'Need to input value.',
    source: source
} );

// plugin 방식 (하기 split 메서드에 regexp 값에 아스테릭스값이 생략되어 있다 /,\s아스테릭스/)

var $xgInputPlugin = $('#xg-input').xgInput({
    width: 150,
    height: 50,
    source: function(query, callbackFunction){
        // 입력 컨트롤의 입력값을 ','로 분리 및 마지막 요소를 분리
        var item = query.split(/,\s/).pop();

        // 쿼리 업데이트
        $('#xg-input').xgInput({query: item});

        callbackFunction(source);
    },
    renderer: function(itemValue, inputValue){
        var inputItems = inputValue.split(/,\s/);

        // 입력 컨트롤의 입력값을 ','로 분리 및 마지막 요소를 분리
        inputItems.pop();

        // 선택된 아이템을 배열에 추가
        inputItems.push(itemValue);

        // 선택된 값을 기존 입력값처럼 ',' 로 추가하기 위해 빈 문자열 요소 추가
        inputItems.push('');

        // 배열요소들을 ', ' 문자열과 병합하여 반환
        var result = inputItems.join(', ');

        return result
    }
} );
