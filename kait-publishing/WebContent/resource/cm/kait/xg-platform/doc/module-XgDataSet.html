XgDataSet

데이터 셋의 인스턴스 생성 및 다중 데이터 테이블의 관리 메서드를 제공한다.

Dependencies

Name Description Remark
jquery DOM Element를 쉽게 관리하는 기능을 제공한다. 1.11.x 이상
json3 JSON <=> 문자열 의 변환 기능을 제공한다.
XgCommon Xg Platform의 공통 상수 및 메서드를 제공한다.
XgDataAdapter 서버와 데이터 통신을 수행하는 메서드를 제공한다.
XgDataTable 데이터 테이블의 인스턴스 생성 및 관리 메서드를 제공한다.

makeListTypeSource(dataTableId, textColumn, valueColumn) → {array}

데이터 테이블의 데이터를 리스트형태의 컴포넌트에서 쓸 수 있도록 오브젝트의 배열을 반환한다. 리스트 형태의 컴포넌트는 콤보박스, 리스트박스, 드롭다운리스트 등이 있다.

Parameters:
Name Type Description
dataTableId string

데이터 테이블의 아이디
textColumn string | number

오브젝트의 text 속성에 쓰일 컬럼이름
valueColumn string | number

오브젝트의 value 속성에 쓰일 컬럼이름
Returns:

text와 value 속성을 가진 오브젝트 배열

Type
array
Example
var source = xgDataSet.makeListTypeSource('dt', 'column_01', 'column_02');
or
var source = xgDataSet.makeListTypeSource('dt', 1, 3);

Parameters:

오브젝트의 id 속성에 쓰일 컬럼이름
textColumn string | number

오브젝트의 text 속성에 쓰일 컬럼이름

select(dataTableIds, cb, params) → {object}

데이터 셋과 연결된 데이터 어댑터를 통하여 각 데이터 테이블에서 사용할 데이터를 서버에 요청한다. xgDataTableIds.length 만큼 select를 수행한다.

Parameters:
Name Type Argument Description
dataTableIds string

서버에서 조회 할 대상 데이터 테이블 아이디
cb function <optional>

콜백 함수
전달 인자 : dataTableIds, result, code
params string | object <optional>

요청시 포함될 파라미터( Serialize 문자열이나 오브젝트 ).
Returns:

xgDataSet

Type
object
Example
var xgDataTableId = 'dt1';

xgDataSet.select(xgDataTableId, function (dataTableIds, result, code) {
    // do something
}, 'key1=value1&key2=value2');

xgDataSet.select(xgDataTableId, null, {foo: 'text', bar: 777});

update(dataTableIds, cb) → {object}

데이터 셋과 연결된 데이터 어댑터를 통하여 데이터 테이블에서 변경된 데이터 서버에 보낸다. 콜백 함수가 생략 되었을 경우, update 요청 후 요청 시 전달된 데이터 테이블 아이디 배열 정보 기준으로 select 를 요청 한다. 만약, DataTable 별로 URL이 지정되어 있을 경우에는 dataTableIds.length만큼 update를 수행하며, 첫번째 DataTable에만 URL이 지정되어 있을 경우에는 update를 한번만 수행한다.

Parameters:
Name Type Argument Description
dataTableIds string

서버에 변경 사항을 반영할 대상 데이터 테이블 아이디 배열
대상 : 추가, 수정, 삭제 된 로우
cb function <optional>

콜백 함수
전달 인자 : dataTableIds, result, code
Returns:

xgDataSet

Type
object
Example
var xgDataTableIds = ['dt1', 'dt2'];

xgDataSet.update(xgDataTableIds, function(dataTableIds, result, code){
    // do something
});
