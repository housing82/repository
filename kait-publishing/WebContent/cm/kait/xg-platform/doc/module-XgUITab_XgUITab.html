XgUITab

XgUITab Class

new XgUITab(selector, option) → {$object}

새로운 탭 인스턴스를 생성한다.

Parameters:
Name Type Argument Description
selector string

CSS 셀렉터
option object <optional>

탭 옵션
Options:
Name Type Default Description
width number｜string null 탭를 감싸는 레이아웃의 폭을 설정한다.
height number｜string null 탭를 감싸는 레이아웃의 높이를 설정한다.
disabled boolean false 비활성화 여부를 설정한다.
rtl boolean false 탭 아이템에 대해 오른쪽 정렬 사용할지 여부를 설정한다.
theme string '' 테마를 설정한다. 테마설정
scrollAnimationDuration number 250 'scrollable'이 true로 설정되어 있을 경우, 좌우로 스크롤 애니메이션에 대한 시간을 설정한다.
enableHover boolean true 탭 아이템에 마우스 오버를 했을 때 hover 기능을 사용할지 여부를 설정한다.
collapsible boolean false 탭 콘텐츠 부분을 접어서 숨기는 기능을 사용할지 여부를 설정한다.
animationType string 'none' 탭 아이템 간의 스위칭 애니메이션의 타입을 설정한다.
[ 'none', 'fade' ]중 택일
enableScrollAnimation boolean true 'scrollable'이 true로 설정되어 있을 경우, 좌우로 스크롤할 경우 애니메이션을 사용할지 여부를 설정한다.
contentTransitionDuration number 450 'anmationType'이 'fade'로 설정되어 있을 경우, 콘텐츠 부분이 나타나는 시간을 설정한다.
toggleMode string 'click' 탭의 아이템 간의 스위칭에 대한 마우스 동작을 설정한다.
[ 'click', 'dblclick', 'mouseenter', 'none' ]중 택일
selectedItem number 0 탭 중 선택되어진 아이템 인덱스를 설정한다.
position string 'top' 탭 컴포넌트의 탭들(탭 헤더)의 위치를 설정한다.
[ 'top', 'bottom' ]중 택일
selectionTracker boolean false 탭 아이템을 선택하였을 때 트래커 애니메이션을 사용할지 여부를 설정한다.
scrollable boolean true 탭 아이템이 많을 경우, 좌우로 스크롤을 사용할지 여부를 설정한다.
scrollPosition string 'right' 스크롤 버튼의 위치를 설정한다.
[ 'left', 'right', 'both' ]중 택일
scrollStep number 70 한번의 스크롤 동작이 이동하는 양(픽셀단위)을 설정한다.
autoHeight boolean true 'height'가 설정되어 있지 않을 경우, 콘텐츠 중 최대 높이만큼 자동으로 높이를 설정한다.
showCloseButtons boolean false 탭 아이템마다 '닫기'버튼을 표시할 지 여부를 설정한다.
closeButtonSize number 16 'showCloseButtons'가 true일 경우, '닫기'버튼의 크기를 설정한다.
initTabContent function null 콘텐츠 패널을 초기화할 때, 수행할 함수를 설정한다.
keyboardNavigation boolean true 탭 아이템 간의 스위칭이나, 선택 등의 키보드 조작 가능 여부를 설정한다.
reorder boolean false 탭 아이템들을 드래그-드롭으로 순서 바꿈 가능 여부를 설정한다.
Listens to Events:
  • created - 탭이 생성되고 초기화 되기 전에 발생한다
  • selected - 탭 아이템 간의 스위칭이 되고 난 후 발생한다
  • tabclick - 탭 아이템이 클릭될 때 발생한다 클릭된 아이템의 인덱스를 반환한다
  • add - 새로운 탭 아이템이 추가되고 난 후 발생한다
  • removed -탭 아이템이 제거되고 난 후 발생한다
  • selecting - 탭 아이템이 선택될 때 발생한다
  • unselecting - 탭 아이템이 선택해제될 때 발생한다
  • unselected - 탭 아이템이 선택해제될 때 발생한다
  • dragStart - 탭 아이템의 드래그 동작이 시작하면 발생한다
  • dragEnd - 탭 아이템의 드래그 동작이 끝나면 발생한다
  • collapsed - 탭 콘텐츠 부분이 접힐 때 발생한다
  • expanded - 탭 콘텐츠 부분이 펴질 때 발생한다
Returns:

$xgTab

Type
$object
Examples
var tabOption = {
    width: 300,
    height: 300,
    ...
};

// module 방식
var $xgTabModule = new XgUITab( '#xg-tab', tabOption );

// plugin 방식
var $xgTabPlugin = $( '#xg-tab' ).xgTab( tabOption );
<!-- data-api 방식 ( Markup Tag로 선언한다. ) -->
<div data-xg-tab
     data-xg-option='{
         "width": 300,
         "height": 300,
         ...
     }'
></div>
