XgUIComboBox

XgUIComboBox Class

new XgUIComboBox(selector, option) → {$object}

새로운 콤보박스 인스턴스를 생성한다.

Parameters:
Name Type Argument Description
selector string

CSS 셀렉터
option object <optional>

콤보박스 옵션
Options:
Name Type Default Description
xgInitOption object {} 데이터 셋 및 데이터 테이블 정보를 설정한다. 자세한 정보는 여기를 참조한다.
xgBindOption object {} 데이터 바인딩 정보를 설정한다. 자세한 정보는 여기를 참조한다.
disabled boolean false 비활성화 여부를 설정한다.
width number｜string null 체크박스를 감싸는 레이아웃의 폭을 설정한다.
height number｜string null 체크박스를 감싸는 레이아웃의 높이를 설정한다.
theme string '' 테마를 설정한다. 테마설정
selectedIndex number -1 source 아이템 중 특정 인텍스의 아이템을 선택된 상태로 설정한다.
multiSelect boolean false 다중선택 모드 활성화 여부를 설정한다.
showArrow boolean true 화살표 버튼( 클릭하면 팝업 리스트를 보여주는 버튼 )을 보여줄지 여부를 설정한다.
showCloseButtons boolean false 선택된 아이템의 닫기 버튼을 보여줄지 여부를 설정한다.
( 다중선택 모드일때 )
validateSelection function null 팝업 리스트에서 아이템 선택 시 유효성 검증을 수행할 함수를 설정한다. 해당 함수는 boolean 값을 반환하여야 하며, 반환값이 false일 경우는 해당 아이템이 선택되지 않는다.
( 다중선택 모드일때 )
전달 인자 : itemValue
searchMode string 'startswith' 입력 컨트롤의 값이 입력되었을 때 소스 데이터와 어떤식으로 비교할지를 설정한다.
  • 'none' : 비교하지 않음
  • 'contains' : 포함( 대소문자 구분 )
  • 'containsignorecase' : 포함
  • 'equals' : 같음( 대소문자 구분 )
  • 'equalsignorecase' : 같음
  • 'startswith' : ~으로 시작( 대소문자 구분 )
  • 'startswithignorecase' : ~으로 시작
  • 'endswith' : ~으로 끝남( 대소문자 구분 )
  • 'endswithignorecase' : ~으로 끝남
autoComplete boolean false 입력 컨트롤의 값이 입력되었을 때 'searchMode'와 부합되는 결과물만 팝업 리스트에 표시할지 여부를 설정한다.
minLength number 2 서버에 요청 시 입력 컨트롤을 통해 사용자에게 입력받을 최소 문자열의 길이를 설정한다. ( 'remoteAutoComplete' 설정이 true 일때 )
source array null 입력 컨트롤의 값을 비교할 로컬 데이터를 설정한다. 배열 안의 요소가 오브젝트일 경우 추가 속성을 지정할 수 있다.
displayMember string '' 'source'안의 아이템 요소가 오브젝트 일 때 화면상에 표시해줄 속성을 설정한다.
valueMember string '' 'source'안의 아이템 요소가 오브젝트 일 때 값으로 사용할 속성을 설정한다.
placeHolder string '' 입력값이 없을 때 보여질 문자열을 설정한다.
popupZIndex number 20000 팝업 리스트의 z-index 를 설정한다.
checkboxes boolean false 팝업 리스트 아이템 요소 옆에 체크박스 기능을 활성화 시킬지 여부를 설정한다.
scrollBarSize number 17 팝업 리스트 내부 스크롤 크기를 설정한다.
enableHover boolean true 팝업 리스트의 아이템에 마우스 오버를 했을 때 hover 기능을 사용할지 여부를 설정한다.
enableSelection boolean true 팝업 리스트의 아이템의 선택가능 여부를 설정한다.
enableBrowserBoundsDetection boolean false 팝업 리스트가 열릴 때 하단에 영역이 충분치 않을 때 팝업이 상단으로 열리는 기능을 활성화할지 여부를 설정한다.
autoOpen boolean false 콤보박스에 마우스를 올렸을 때 자동으로 팝업 리스트가 열리는 기능의 활성화 여부를 설정한다.
dropDownHorizontalAlignment string 'left' 팝업 리스트의 위치를 입력 컨트롤 기준으로 어느쪽에 기준을 두고 표시될지 설정한다.
[ 'left', 'right' ] 중 택일
dropDownHeight number 200 팝업 리스트의 높이를 설정한다.
dropDownWidth number 200 팝업 리스트의 폭을 설정한다.
autoDropDownHeight boolean false 팝업 리스트의 높이를 내부 아이템의 갯수에 맞게 자동 조절되는 기능을 활성화 여부를 설정한다.
itemHeight number -1 팝업 리스트 아이템들의 높이를 설정한다.
renderer function null 팝업 리스트 아이템들을 어떻게 렌더링할지 설정한다. 해당 함수의 반환값이 렌더링된다.
전달 인자 : itemIndex, label, text
renderSelectedItem function null 팝업 리스트에서 선택된 아이템을 어덯게 렌더링할지 설정한다. 해당 함수의 반환값이 렌더링된다.
전달 인자 : itemIndex, item
openDelay number 350 팝업 리스트가 열리기 까지의 지연시간을 설정한다.
closeDelay number 400 팝업 리스트가 딛히기 까지의 지연시간을 설정한다.
animationType string 'slide' 팝업 리스트의 에니메이션을 설정한다.
[ 'slide', 'fade', 'none' ] 중 택일

xgInitOption

Name Type Default Description
xgDataSet object|string null 데이터 셋을 설정한다. 데이터 셋 오브젝트 또는 데이터 셋 아이디를 지정할 수 있다.
xgDataTable object|string null 데이터 테이블을 설정한다. 데이터 테이블 오브젝트 또는 데이터 테이블 아이디를 지정할 수 있다.
해당 데이터 테이블은 같이 지정된 데이터 셋에 추가된 상태여야 한다.
xgBindOption

Name Type Default Description
mode string 'read' 데이터 바인딩 모드를 설정한다. 'bind.type' 이 'row' 일 경우에는 'read' 모드만 지원된다.
[ 'read', 'write' ] 중 택일
  • 'read' : 해당 컴퍼넌트의 변경사항이 타 컴퍼넌트에게 전달되지 않는다.
  • 'write' : 해당 컴퍼넌트의 변경사항이 타 컴퍼넌트에게 전달된다.
sync boolean true 데이터 테이블 로우의 상태값 및 제약조건의 반영 여부를 설정한다.
  • true : 로우의 상태값 및 제약조건과 상호작용( 자동 비활성화 ) 기능을 사용한다. 선택된 로우가 삭제될 경우 수정불가능 상태로 변경된다. 또한 서버와 동기화 된 'key', 'sequence' 컬럼을 바라보는 경우 수정불가능 상태로 변경된다.
  • false : 로우의 상태값 및 제약조건과 상호작용( 자동 비활성화 ) 기능을 사용하지 않는다.
bind object {} 오브젝트 구조는 여기를 참조한다.
bind object

Name Type Default Description
type string 'column' 데이터 테이블의 바인딩 목표를 설정한다. 데이터 테이블의 현재 선택된 로우를 기준으로 동작된다.
'column' 모드만 지원된다.
column string|number null 'column' type으로 설정되었을 때 바인딩 할 컬럼을 지정한다. 컬럼의 이름 또는 컬럼의 인덱스로 지정할 수 있다. ( 데이터 테이블 기준 )
item object in source array

요소에 추가된 아이템은 하기의 속성을 지닌 오브젝트로 변환되어 지정된다.

Name Description
label 리스트에 보여질 'displayMember'
value 실제 값으로 취급될 'valueMember'
disabled 아이템의 비활성화 여부
checked 아이템의 체크상태 여부
hasThreeStates 아이템의 체크상태를 3단계로 구분지어진지 여부
html 아이템의 보여지는 마크업
group 아이템의 그룹정보
Listens to Events:
  • select - 팝업 리스트의 아이템을 선택될 때 발생한다
  • unselect - 팝업 리스트의 아이템이 선택해제될 때 발생한다
  • change - 팝업 리스트의 선택된 아이템이 변경될 때 발생한다
  • open - 팝업 리스트가 열릴 때 발생한다
  • close - 팝업 리스트가 닫힐 때 발생한다
  • checkChange - 팝업 리스트의 아이템의 체크상태가 변경될 때 발생한다( 'checkboxes' 설정이 true 일 때 )
  • bindingComplete - 'source' 콤보박스에 설정이 완료될 때 발생한다
Returns:

$xgComboBox

Type
$object
Examples
var comboBoxOption = {
    xgInitOption: {
        xgDataSet: dataSet,
        xgDataTable: 'dataTableName'
    },
    xgBindOption: {
        mode: 'write',
        bind: {
            type: 'column',
            column: 'columnName'
        }
    },
    width: 300,
    height: 300,
    ...
    source: [ { text: 'foo', value: 0 }, { text: 'bar', value: 1 } ]
    displayMember: 'text'
    valueMember: 'value'
};

// module 방식
var $xgComboBoxModule = new XgUIComboBox( '#xg-combo-box', comboBoxOption );

// plugin 방식
var $xgComboBoxPlugin = $( '#xg-combo-box' ).xgComboBox( comboBoxOption );
