XgUITree

계층 구조를 가진 데이터를 표현하는 기능을 제공하는 컴포넌트이다.

Dependencies

Name Description Remark
jquery DOM Element를 쉽게 관리하는 기능을 제공한다. 1.11.x 이상
XgUICommon Xg UI Components의 공통 상수 및 메서드를 제공한다.

트리 옵션

xgTree('ensureVisible', element)

특정 아이템이 트리의 크기나 아이템의 순서에 상관없이 지정된 아이템의 위치로 이동하여 보여질 수 있도록 설정한다.

Parameters:
Name Type Description
'ensureVisible' string

메서드 이름
element unknown

대상 아이템 요소(LI tag)
Example
var element = $(selector)[0];
$xgTree.xgTree('ensureVisible', element);

선택된 아이템을 반환한다. jQuery의 val() 메소드를 이용해도 된다. 두번째 파라메터가 전달될 경우, 특정 아이템의 요소의 아이템을 선택한다.

Parameters:
Name Type Argument Description
'val' string

메서드 이름
element unknown <optional>

대상 아이템 요소(LI tag)
Returns:

value

Type
unknown
Example
var value = $xgTree.xgTree('val');

$xgTree.xgTree('val', element);

// jQuery 함수 이용
var value = $xgTree.val();

$xgTree.val(element);
