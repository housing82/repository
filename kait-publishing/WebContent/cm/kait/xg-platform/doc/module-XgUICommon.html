XgUICommon

UI Components에서 사용하는 상수 및 메서드를 제공한다.

Dependencies

Name Description Remark
jquery DOM Element를 쉽게 관리하는 기능을 제공한다. 1.11.x 이상
swidgets UI Component의 기본 기능을 제공한다.
XgCommon Xg Platform의 공통 상수 및 메서드를 제공한다.

Members

<static, constant> constant

공통으로 사용하는 상수

Name Description
GRID.TEMPLATE 그리드 템플릿 옵션
  • ALL : 조회 및 수정에 대한 전반적인 옵션을 제공한다.
    1. contextMenu.rowControl
    2. contextMenu.export
    3. cellEdit
    4. cellValidation
    5. columnFormat
    6. showRowStatus
    7. useEditor
  • VIEW : 조회에 특화된 옵션을 제공한다.
    1. contextMenu.export
    2. columnFormat
    3. showRowStatus
GRID.MARKUP 그리드에서 사용하는 마크업 템플릿
Example
var allTemplateOption = XgUICommon.constant.GRID.TEMPLATE.ALL;

var viewTemplateOption = XgUICommon.constant.GRID.TEMPLATE.VIEW;

<static, constant> localization

다국어 메시지 오브젝트

Name Description
en 영어 [ default ]
ko 한국어
Example
var en = XgUICommon.localization.en;

var ko = XgUICommon.localization.ko;

<static> changeStrStyle(str, method, caseStr, dataApiYn) → {string}

변환 대상 문자열을 메서드를 지정하여 변환하여 반환한다. 추가 인자를 통하여 대소문자 변환 및 data-api 방식으로 사용가능 한 문자열을 만들 수 있다.

Parameters:
Name Type Argument Description
str string

변환 대상 문자열
method string

문자열 변환 메서드 이름
[ 'camelToDash', 'dashToCamel' ] 중의 값
caseStr string <optional>

대소문자 변환 대상
[ 'upper', 'lower' ] 중의 값
dataApiYn boolean <optional>

data-api 방식으로 만들지 여부
Returns:

str

Type
string
Example
var str = 'xg-foo-bar'

var camelStr = XgUICommon.changeStrStyle( str, 'dashToCamel' );

var camelUpperStr = XgUICommon.changeStrStyle( str, 'dashToCamel', 'upper' );

var dashDataApiStr = XgUICommon.changeStrStyle( 'xgFooBar', 'camelToDash', null, true );

<static> getLocalization(localeCode) → {object}

국가 코드에 매칭되는 국제화 메시지 오브젝트를 반환한다. 국가 코드가 생략 되었을 경우 브라우저의 국가 코드를 이용한다. 해당 코드의 메시지 오브젝트가 없을 경우 기본 언어 ( 한글 )의 메시지 오브젝트가 반환한다.

Parameters:
Name Type Argument Description
localeCode string <optional>

국가 코드
Returns:

localization

Type
object
Example

var enLocalization = XgUICommon.getLocalization( 'ko' );

<static> setBindColInfo(xgDataTable, opt) → {object}

옵션 오브젝트의 bind.column 의 데이터 타입을 체크하여 해당 옵션의 bind 속성에 컬럼의 이름과 인덱스 값을 재설정 하여 반환한다.

Parameters:
Name Type Description
xgDataTable object

데이터 테이블 오브젝트
opt object

바인딩 옵션이 포함된 오브젝트
Returns:

opt

Type
object
Example
// xgDataTable 인스턴스가 이미 존재하고, 0번째 컬럼이름이 'foo'라고 지정되어 있다고 가정한다.
var optWithColumnIndex = {
    bind: {
        column: 0
    }
};

XgUICommon.setBindColInfo( xgDataTable, optWithColumnIndex );

var optWithColumnName = {
    bind: {
        column: 'foo'
    }
};

XgUICommon.setBindColInfo( xgDataTable, optWithColumnName );

<static> setXgDataTable(obj, opt) → {object}

옵션으로 데이터 테이블 참조 정보가 문자열인 경우 데이터 셋에서 해당 데이터 테이블 오브젝트를 설정한다.

Parameters:
Name Type Description
obj object

오브젝트
opt object

데이터 셋과 데이터 테이블 참조 정보( 이름 또는 오브젝트 )가 담긴 오브젝트
Returns:

xgDataTable

Type
object
Example
// xgDataSet에 'foo'라는 데이터 테이블이 존재한다고 가정한다.
var xgDataTable = {};

var opt = {
    xgDataSet: xgDataSet
    xgDataTable: 'foo'
};

XgUICommon.setXgDataTable( xgDataTable, opt );
