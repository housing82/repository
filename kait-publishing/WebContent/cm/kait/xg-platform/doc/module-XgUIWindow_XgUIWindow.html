XgUIWindow

XgUIWindow Class

new XgUIWindow(selector, option) → {$object}

새로운 윈도우 인스턴스를 생성한다.

Parameters:
Name Type Argument Description
selector string

CSS 셀렉터
option object <optional>

윈도우 옵션
Options:
Name Type Default Description
width number/string null 윈도우를 감싸는 레이아웃의 폭을 설정한다.
height number/string null 윈도우를 감싸는 레이아웃의 높이를 설정한다.
disabled boolean false 비활성화 여부를 설정한다.
theme string '' 테마를 설정한다. 테마설정
collapsed boolean false 윈도우를 접혀진 상태로 설정한다.
collapseAnimationDuration number 150 윈도우가 펼쳐질 때의 애니메이션에 대한 시간을 설정한다.
minHeight number/string 50 윈도우를 감싸는 레이아웃의 최소 높이를 설정한다.
maxHeight number/string 600 윈도우를 감싸는 레이아웃의 최대 높이를 설정한다.
minWidth number/string 50 윈도우를 감싸는 레이아웃의 최소 폭을 설정한다.
maxWidth number/string 600 윈도우를 감싸는 레이아웃의 최대 폭을 설정한다.
showCloseButton boolean true 윈도우의 타이틀바에 닫힘 버튼을 보일지에 대한 여부를 설정한다.
showCollapseButton boolean false 윈도우의 타이틀바에 접힘/펼침 버튼을 보일지에 대한 여부를 설정한다.
autoOpen boolean true 윈도우가 생성이 됨과 동시에 보여질지에 대한 여부를 설정한다.
keyboardCloseKey number/string 'esc' 윈도우 닫기에 사용할 키의 키값을 설정한다. ESC키는 'esc'로 설정가능하며, 기본값으로 설정되어 있다.
keyboardNavigation boolean true 키보드를 사용하여 윈도우를 조작 가능 여부를 설정한다.
title string '' 윈도우에 나타낼 타이틀을 설정한다.
content string '' 윈도우의 콘텐츠 중 HTML 콘텐츠를 설정한다.
initContent method 윈도우의 콘텐츠를 초기화하는 함수를 설정한다.
draggable boolean true 윈도우의 드래그 이동 가능 여부를 설정한다.
resizable boolean true 윈도우 크기의 재조정 가능 여부를 설정한다.
animationType string 'fade' 윈도우가 열리고 닫힐 때의 애니메이션 타입을 설정한다.
[ 'none', 'fade', 'slide', 'combined' ]중 택일
closeAnimationDuration number 350 윈도우가 닫힐 때의 애니메이션에 대한 시간을 설정한다.
showAnimationDuration number 350 윈도우가 보여질 때의 애니메이션에 대한 시간을 설정한다.
isModal boolean false 윈도우가 모달 다이얼로그처럼 동작할지에 대한 여부를 설정한다.
position object/string 'center' 윈도우의 위치를 설정한다.
'center', 'top, left', '{x:300, y:500}', '[300, 500']' 등의 형태로 설정 가능
closeButtonSize number 16 윈도우 닫기 버튼의 크기를 설정한다.
colseButtonAction string 'hide' 윈도우 닫기 버튼을 눌렀을 때의 동작을 설정한다.
[ 'hide', 'close' ]중 택일
modalOpacity number 0.3 윈도우가 모달 다이얼로그 모드로 동작할 때, 배경의 불투명도를 설정한다.
modalZIndex number 18000 윈도우가 모달 다이얼로그 모드로 동작할 때의 Z-index를 설정한다.
modalBackgroundZIndex number 12990 윈도우가 모달 다이얼로그 모드로 동작할 때의 백그라운드 z-index를 설정한다.(설명 보강 필요)
zIndex nuber 9001 윈도우의 z-index를 설정한다.
dragArea object null 윈도우를 드래그나 이동할 수 있는 스크린 영역을 설정한다.
{left:100, top:100, width:600, height:600}와 같은 식으로 설정
okButton object null 윈도우의 '확인'/'OK' 버튼을 설정한다.
cancelButton object null 윈도우의 '취소'/'Cancel' 버튼을 설정한다.
Listens to Events:
  • created - 윈도우가 생성될 때 발생한다
  • moving - 윈도우를 움직일 때 발생한다
  • moved - 윈도우를 움직인 후에 발생한다
  • open - 윈도우를 열 때 발생한다
  • close -윈도우를 닫을 때 발생한다
  • expand - 윈도우를 펼칠 때 발생한다
  • collapse - 윈도우를 접을 때 발생한다
  • resizing - 윈도우의 크기를 조정할 때 발생한다
  • resized - 윈도우의 크기를 조정한 후에 발생한다
Returns:

$xgWindow

Type
$object
Examples
var windowOption = {
    width: 300,
    height: 300,
	   resizable: false,
	   okButton: $('#okButton'),
	   cancelButton: $('#cancelButton'),
    ...
};

// module 방식
var $xgWindowModule = new XgUIWindow( '#xg-window', windowOption );

// plugin 방식
var $xgWindowPlugin = $( '#xg-window' ).xgWindow( windowOption );
<!-- data-api 방식 ( Markup Tag로 선언한다. ) -->
<div data-xg-window
     data-xg-option='{
         "width": 300,
         "height": 300,
	        "resizable": false,
         "okButton": "$('#okButton')",
	        "cancelButton": "$('#cancelButton')",
         ...
     }'
></div>
