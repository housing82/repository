XgUIDateTimeInput

XgUIDateTimeInput Class

new XgUIDateTimeInput(selector, option) → {$object}

새로운 날짜 입력 컨트롤 인스턴스를 생성한다.

Parameters:
Name Type Argument Description
selector string

CSS 셀렉터
option object <optional>

달력 옵션
Options:
Name Type Default Description
xgInitOption object null 데이터 셋 및 데이터 테이블 정보를 설정한다. 자세한 정보는 여기를 참조한다.
xgBindOption object null 데이터 바인딩 정보를 설정한다. 자세한 정보는 여기를 참조한다.
disabled boolean false 비활성화 여부를 설정한다.
min date Date( 1990, 1, 1 ) 선택가능 한 최소일자를 설정한다.
max date Date( 2100, 1, 1 ) 선택가능 한 최대일자를 설정한다.
formatString string 'dd/MM/yyyy' 입력 컨트롤의 포멧을 설정한다.
ISO 8601 Date Format
width number｜string null 폭을 설정한다.
height number｜string null 높이를 설정한다.
popupZIndex number 20000 팝업의 z-index 를 설정한다.
showCalendarButton boolean true 달력 버튼 표시여부를 설정한다.
allowNullDate boolean true null 값을 허용할지 여부를 설정한다.
allowKeyboardDelete boolean true 입력 컨트롤의 값을 키보드의 delete 키로 삭제를 허용할지 여부를 설정한다.
theme string '' 테마를 설정한다. 테마 설정
value Date Today 날짜를 설정한다.
firstDayOfWeek number 0 한주의 시작으로 지정할 요일을 설정한다.
  • 0 : 일요일
  • 1 : 월요일
  • 2 : 화요일
  • 3 : 수요일
  • 4 : 목요일
  • 5 : 금요일
  • 6 : 토요일
showWeekNumbers boolean false 주차를 표시할지 여부를 설정한다.
textAlign string 'left' 입력 컨트롤의 정렬방식을 설정한다.
[ 'left', 'center', 'right' ] 중 택일
enableViews boolean true 달력의 상단 바를 눌렀을 때 달 또는 년도를 선택하는 기능의 활성화 여부를 설정한다.
rowHeaderWidth number 25 주차 표시영역의 폭을 설정한다. ( 'showWeekNumbers' 가 true 설정되어 있을 때 )
columnHeaderHeight number 20 주차 표시영역의 높이를 설정한다. ( 'showWeekNumbers' 가 true 설정되어 있을 때 )
titleHeight number 25 상단 바의 높이를 설정한다.
dayNameFormat string 'firstTwoLetters' 요일의 포멧을 설정한다.
  • 'default' : 3글자로 표시한다.
  • 'shortest' : 2글자( 축약어 )로 표시한다.
  • 'firstTwoLetter' : 앞의 2글자로 표시한다.
  • 'firstLetter' : 앞의 1글자로 표시한다.
  • 'full' : 축약없이 본래의 값을 보여준다.
readOnly boolean false 읽기 모드로 사용할지 여부를 설정한다.
showFooter boolean false 'Today' 및 'Clear' 기능이 적용된 하단 영역화면의 표시여부를 설정한다.
selectionMode string 'default' 날짜 선택 모드를 설정한다.
  • 'none' : 날짜를 클릭해도 선택되지 않는다.
  • 'default' : 클릭할 때 마다 선택된 날짜가 변경된다.
  • 'range' : 첫번째 클릭 날짜 부터 두번째 클릭 날짜까지 다중 선택된다.
todayString string 'Today' 하단 레이아웃의 Today 표시 값을 설정한다. ( 'showFooter' 가 true 설정되어 있을 때 )
clearString string 'Clear' 하단 레이아웃의 Clear 표시 값을 설정한다. ( 'showFooter' 가 true 설정되어 있을 때 )
culture string 'en-US' 달력의 언어셋을 설정한다. 설정전에 언어별 스크립트를 달력 컴퍼넌트가 로드 후 추가 로드하여야 한다.
eg ) 'ko-KR' ( 한국어 언어셋 설정 시 )
openDelay number 350 팝업 달력이 열리기 까지의 지연시간을 설정한다.
closeDelay number 40 팝업 달력이 닫히기 까지의 지연시간을 설정한다.
closeCalendarAfterSelection boolean true 달력에서 날짜를 선택했을 때 달력을 닫을지 여부를 설정한다.
enableBrowserBoundsDetection boolean false 팝업 달력이 열릴 때 하단에 영역이 충분치 않을 때 팝업이 상단으로 열리는 기능을 활성화할지 여부를 설정한다.
dropDownHorizontalAlignment string 'left' 팝업 달력의 위치를 입력 컨트롤 기준으로 어느쪽에 기준을 두고 표시될지 설정한다.
[ 'left', 'right' ] 중 택일
animationType string 'slide' 팝업 달력의 에니메이션을 설정한다.
[ 'slide', 'fade', 'none' ] 중 택일
enableAbsoluteSelection boolean false 사용자가 입력 컨트롤에 포커스를 위치했을 때 첫번째 심볼만 선택되는 기능을 적용할지 여부를 설정한다.
calendar object null 달력의 속성 등을 설정한다.

xgInitOption

Name Type Default Description
xgDataSet object|string null 데이터 셋을 설정한다. 데이터 셋 오브젝트 또는 데이터 셋 아이디를 지정할 수 있다.
xgDataTable object|string null 데이터 테이블을 설정한다. 데이터 테이블 오브젝트 또는 데이터 테이블 아이디를 지정할 수 있다.
해당 데이터 테이블은 같이 지정된 데이터 셋에 추가된 상태여야 한다.
xgBindOption

Name Type Default Description
mode string 'read' 데이터 바인딩 모드를 설정한다. 'bind.type' 이 'row' 일 경우에는 'read' 모드만 지원된다.
[ 'read', 'write' ] 중 택일
  • 'read' : 해당 컴퍼넌트의 변경사항이 타 컴퍼넌트에게 전달되지 않는다.
  • 'write' : 해당 컴퍼넌트의 변경사항이 타 컴퍼넌트에게 전달된다.
sync boolean true 데이터 테이블 로우의 상태값 및 제약조건의 반영 여부를 설정한다.
  • true : 로우의 상태값 및 제약조건과 상호작용( 자동 비활성화 ) 기능을 사용한다. 선택된 로우가 삭제될 경우 수정불가능 상태로 변경된다. 또한 서버와 동기화 된 'key', 'sequence' 컬럼을 바라보는 경우 수정불가능 상태로 변경된다.
  • false : 로우의 상태값 및 제약조건과 상호작용( 자동 비활성화 ) 기능을 사용하지 않는다.
immediately boolean false 'write' 모드 일 때 변경사항을 즉시 전파할지 여부를 설정한다. false로 설정할 경우 컴퍼넌트에서 포커스를 잃을 때 변경사항을 전파한다.
bind object {} 오브젝트 구조는 여기를 참조한다.
bind object

Name Type Default Description
type string 'column' 데이터 테이블의 바인딩 목표를 설정한다. 데이터 테이블의 현재 선택된 로우를 기준으로 동작된다.
'column' 모드만 지원된다.
column string|number null 'column' type으로 설정되었을 때 바인딩 할 컬럼을 지정한다. 컬럼의 이름 또는 컬럼의 인덱스로 지정할 수 있다. ( 데이터 테이블 기준 )
Listens to Events:
  • valueChanged - 값이 변경될 때 발생한다
  • change - 값이 변경된 후 포커스를 잃을 때 발생한다
  • textchanged - 입력 컨트롤의 텍스트가 변경될 때 발생한다
  • open - 팝업 달력이 열릴 때 발생한다
  • close - 팝업 달력이 닫힐 때 발생한다
Returns:

$xgDateTimeInput

Type
$object
Examples
var dateTimeInputOption = {
    xgInitOption: {
        xgDataSet: dataSet,
        xgDataTable: 'dataTableName'
    },
    xgBindOption: {
        mode: 'read',
        immediately: true,
        bind: {
            type: 'column',
            column: 'columnName'
        }
    },
    calendar: {
        titleFormat: ['yyyy년 M월','yyyy년','yyyy년']
    },
    min: new Date( 2015, 1, 1 ),
    max: new Date( 2025, 12, 31 ),
    width: 300,
    height: 300,
    ...
    firstDayOfWeek: 1,
    showWeekNumbers: true,
    culture: 'ko-KR'
};

// module 방식
var $xgDateTimeInputModule = new XgUIDateTimeInput( '#xg-date-time-input', dateTimeInputOption );

// plugin 방식
var $xgDateTimeInputPlugin = $( '#xg-date-time-input' ).xgDateTimeInput( dateTimeInputOption );
