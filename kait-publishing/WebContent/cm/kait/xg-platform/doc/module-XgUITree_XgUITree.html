XgUITree

XgUITree Class

new XgUITree(selector, option) → {$object}

새로운 트리 인스턴스를 생성한다.

Parameters:
Name Type Argument Description
selector string

CSS 셀렉터
option object <optional>

트리 옵션
Options:
Name Type Default Description
width number｜string null 트리를 감싸는 레이아웃의 폭을 설정한다.
height number｜string null 트리를 감싸는 레이아웃의 높이를 설정한다.
disabled boolean false 비활성화 여부를 설정한다.
theme string '' 테마를 설정한다. 테마설정
animationShowDuration number 350 트리가 펼쳐질 때의 애니메이션에 대한 시간을 설정한다.
animationHideDuration number 350 트리가 접혀질 때의 애니메이션에 대한 시간을 설정한다.
enableHover boolean false 트리의 아이템에 마우스 오버를 했을 때 hover 기능을 사용할지 여부를 설정한다.
keyboardNavigation boolean true 트리 내에서 방향키 등의 키보드 조작을 사용할지 여부를 설정한다.
toggleMode string 'dblclick' 트리의 아이템을 접고 펼치는 동작에 대한 마우스 조작을 설정한다.
[ 'click', 'dblclick' ]중 택일
source array null 트리의 아이템을 구성할 로컬 데이터를 설정한다. 배열안의 요소는 오브젝트로 구성이 되면, 구성 요소를 참고하여 구성한다.
incrementalSearch boolean true 아이템에 대한 증분 검색을 사용할지 여부를 설정한다.
checkboxes boolean false 트리의 아이템마다 체크박스를 사용할지 여부를 설정한다.
hasThreeStates boolean false 체크박스를 사용할 경우, 하위 아이템이 일부만 체크되면 'indeterminate' 되도록 설정한다.
allowDrag boolean false 트리의 아이템들을 드래그할 수 있도록 허용할지 여부를 설정한다. allowDrop과 함께 사용하는 것을 권장한다.
allowDrop boolean true 트리의 아이템들을 드래그하여 드롭으로 위치이동하는 것을 허용할지 여부를 설정한다. allowDrag와 함께 사용하는 것을 권장한다.
dragStart function null 트리의 아이템을 드래그 시작할 때, 수행할 함수를 설정한다.
dragEnd function null 트리의 아이템을 드래그를 마칠 때, 수행할 함수를 설정한다.
toggleIndicatorSize number 16 접고 펼치기 위한 화살표의 크기를 설정한다.

item object in source array

트리의 아이템은 아래의 속성을 지닌 오브젝트로 변환되어 지정된다.

Name Description
label 트리에서 보여질 아이템의 이름
value 실제 값으로 취급될 속성
html 아이템에서 표현되어질 마크업
id 트리에서 아이템을 구분할 때 쓰여질 ID
disabled 아이템의 비활성화에 대한 여부
checked 체크박스를 가질 경우, 아이템의 체크상태에 대한 여부
expanded (하위 아이템을 가지고 있는) 아이템이 펼쳐진 상태로 보여질 지에 대한 여부
selected (하위 아이템을 가지고 있는) 아이템이 선택된 상태로 보여질 지에 대한 여부
items 아이템이 가지는 하위 아이템에 대한 배열
icon 아이템의 아이콘(URL 형태를 가질 수 있다.)
iconsize 아이템의 아이콘 사이즈

item object in tree

트리에 존재하는 아이템은 아래의 속성을 지닌 오브젝트로 변환되어 반환된다.

Name Description
label 트리에서 보여지는 아이템의 이름
value 실제 값으로 취급되는 속성
disabled 아이템의 비활성화에 대한 여부
checked 체크박스를 가질 경우, 아이템의 체크 상태에 대한 여부
element 아이템의 요소(LI tag)
parentElement 아이템의 부모 아이템 요소
isExpanded 아이템이 펼쳐지거나 접혀진 상태에 대한 여부
selected 아이템이 선택 상태에 대한 여부
Listens to Events:
  • expand - 트리의 아이템을 펼칠 때 발생한다
  • collapse - 트리의 아이템을 접을 때 발생한다
  • select - 트리의 아이템을 선택할 때 발생한다
  • initialized - 트리를 초기화할 때 발생한다
  • added -트리에 아이템을 추가할 때 발생한다
  • removed - 트리에서 아이템을 제거할 때 발생한다
  • checkChange - 트리의 아이템들의 체크상태가 변경될 때 발생한다
  • dragStart - 트리의 아이템을 드래그 시작할 때 발생한다
  • dragEnd - 트리의 아이템을 드래그 완료할 때 발생한다
Returns:

$xgTree

Type
$object
Examples
var treeOption = {
    width: 300,
    height: 300,
    ...
    source: [ { label: 'foo', value: 0 }, { label: 'bar', value: 1 } ]
};

// module 방식
var $xgTreeModule = new XgUITree( '#xg-tree', treeOption );

// plugin 방식
var $xgTreePlugin = $( '#xg-tree' ).xgTree( treeOption );
<!-- data-api 방식 ( Markup Tag로 선언한다. ) -->
<div data-xg-tree
     data-xg-option='{
         "width": 300,
         "height": 300,
         ...
         "source": [ {"label": "foo", "value": 0 }, { "label": "bar", "value": 1 } ]
     }'
></div>
