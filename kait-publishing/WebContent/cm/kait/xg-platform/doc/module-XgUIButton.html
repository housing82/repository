XgUIButton

기본 버튼 기능 외에 토글, 스위치, 드랍다운, 링크 등의 추가 기능을 제공한다.

Dependencies

Name Description Remark
jquery DOM Element를 쉽게 관리하는 기능을 제공한다. 1.11.x 이상
XgUICommon Xg UI Components의 공통 상수 및 메서드를 제공한다.

xgButton('val', newValue) → {string}

버튼의 value값을 반환한다. 2번째 인자가 전달된 경우 값을 반환하지 않고 버튼의 value값을 변경한다. jQuery에서 직접 제공되는 val() 메소드를 이용해도 된다.

Parameters:
Name Type Argument Description
'val' string

메서드 이름
newValue string | number <optional>

새로 지정할 값
Returns:

value

Type
string
Example
$button.xgButton('val', 'foo');

var fooValue = $button.xgButton('val');

// jQuery 함수 이용
$button.val('bar');

var barValue = $button.val();

xgButton(option) → {object}

버튼에 옵션을 적용 시킨다. 모듈 방식으로 생성 시 첫번째 인자로 'default' 문자열을 전달하여야 한다.
버튼을 적용 시킬 DOM은 input[type="button"] 또는 button 엘리먼트 이여야 한다.
하기 옵션은 xgButton, xgToggleButton, xgRepeatButton, xgLinkButton 에서 공통으로 사용 가능하다.

Parameters:
Name Type Description
option object

버튼 옵션
Options:
Name Type Default Description
roundedCorners string 'all' 외각 모서리를 곡선 처리할 대상을 지정한다.
[ 'all', 'top', 'bottom', 'left', 'right', 'top-right', 'top-left', 'bottom-right', 'bottom-left' ] 중 택일
disabled boolean false 버튼의 활성화 상태를 설정한다.
height number｜string null 높이를 설정한다. ( 문자열일 경우 px, % 단위를 지원한다. )
width number｜string null 폭을 설정한다. ( 문자열일 경우 px, % 단위를 지원한다. )
theme string '' 테마를 설정한다. 테마 지정
template string 'default' 템플릿을 설정한다.
[ 'default', 'primary', 'success', 'warning', 'danger', 'inverse', 'info', 'link' ] 중 택일
Listens to Events:
  • click - 버튼을 클릭할 때 발생한다
Returns:

$xgButton

Type
object
Example
var buttonOption = {
    width: 150,
    height: 25
};

var $xgButtonPlugin = $('#xg-button').xgButton(buttonOption);

var $xgButtonModule = new XgUIButton('default', '#xg-button', buttonOption);

// 설정 한 옵션 값을 구하려면 옵션의 이름 문자열을 인자로 넘겨준다.
// $xgButtonModule.xgButton('optionName');

var widthOptionValue = $xgButtonModule.xgButton('width');

$xgButtonModule.on('click', function(event) {
    // do something..
});

xgButtonGroup(option) → {object}

버튼 그룹에 옵션을 적용 시킨다. 모듈 방식으로 생성 시 첫번째 인자로 'buttonGroup' 문자열을 전달하여야 한다.
드랍다운 버튼을 적용 시킬 DOM은 버튼 DOM을 감싸고 있는 div 엘리먼트 이여야 한다.

Parameters:
Name Type Description
option object

버튼 그룹 옵션
Options:
Name Type Default Description
disabled boolean false 버튼 그룹의 활성화 상태를 설정한다.
비활성화 상태라도 버튼에 전달되는 이벤트 전파를 가로막지 않는다.
enableHover boolean false 마우스를 올렸을 때 버튼에 hover 효과를 보여줄지 설정한다.
theme string '' 테마를 설정한다. 테마 지정
mode string 'default' 버튼 그룹이 동작할 방식을 설정한다.
[ 'checkbox', 'radio', 'default' ] 중 택일
To Do:
  • disabled 옵션의 경우 비활성화 상태로 설정 시 버튼 이벤트가 전파되는 문제가 있다
Listens to Events:
  • buttonclick - 버튼 그룹 내부의 버튼이 클릭될 때 발생한다
  • selected - 버튼이 선택 될 때 발생한다 ( checkbox 및 radio 모드만 지원 )
  • unselected - 버튼이 선택해제 될 때 발생한다 ( checkbox 및 radio 모드만 지원 )
Returns:

$xgButtonGroup

Type
object
Example
var buttonOption = {
    mode: 'radio'
};

var $xgButtonGroupPlugin = $('#xg-button-group').xgButtonGroup(buttonOption);

var $xgButtonGroupModule = new XgUIButton('buttonGroup', '#xg-button-group', buttonOption);

$xgButtonGroupPlugin.on('buttonClick', function(event){
    // do something..
});

$xgButtonGroupModule.on('selected', function(event){
    // do something..
});

xgSwitchButton('val', newValue) → {string}

버튼의 value값을 반환한다. 2번째 인자가 전달된 경우 값을 반환하지 않고 버튼의 value값을 변경한다. jQuery에서 직접 제공되는 val() 메소드를 이용해도 된다.

Parameters:
Name Type Argument Description
'val' string

메서드 이름
newValue boolean <optional>

새로 지정할 값
Returns:

value

Type
string
Example
$switch.xgSwitchButton('val', true);

var trueValue = $switch.xgSwitchButton('val');

// jQuery 함수 이용
$switch.val(false);

var falseValue = $switch.val();

토글 버튼에 옵션을 적용 시킨다. 모듈 방식으로 생성 시 첫번째 인자로 'toggle' 문자열을 전달하여야 한다.

Parameters:
Name Type Description
option object

토글 버튼 옵션
Options:
Name Type Default Description

See:
  • 버튼 공통 옵션
Listens to Events:
  • click - 버튼을 클릭할 때 발생한다

$xgToggleButton

object

    // do something..
