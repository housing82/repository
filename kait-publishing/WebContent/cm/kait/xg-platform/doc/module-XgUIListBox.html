XgUIListBox

배열 아이템을 나열하고 선택가능한 기능을 제공하며 콤보박스의 팝업 리스트의 기반 컴퍼넌트이다.

Dependencies

Name Description Remark
jquery DOM Element를 쉽게 관리하는 기능을 제공한다. 1.11.x 이상
XgCommon Xg Platform의 공통 상수 및 메서드를 제공한다.
XgUICommon Xg UI Components의 공통 상수 및 메서드를 제공한다.
