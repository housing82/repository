XgUIMaskedInput

XgUIMaskedInput Class

new XgUIMaskedInput(selector, option) → {$object}

새로운 마스크 입력 컨트롤의 인스턴스를 생성한다.

Parameters:
Name Type Argument Description
selector string

CSS 셀렉터
option object <optional>

마스크 입력 컨트롤 옵션
Options:
Name Type Default Description
xgInitOption object {} 데이터 셋 및 데이터 테이블 정보를 설정한다. 자세한 정보는 여기를 참조한다.
xgBindOption object {} 데이터 바인딩 정보를 설정한다. 자세한 정보는 여기를 참조한다.
value string null 마스크 입력 컨트롤의 값을 설정한다.
mask string '#####' 마스크를 설정한다.
disabled boolean false 비활성화 여부를 설정한다.
width number｜string 300 마스크 입력 컨트롤을 감싸는 레이아웃의 폭을 설정한다.
height number｜string 35 마스크 입력 컨트롤을 감싸는 레이아웃의 높이를 설정한다.
theme string '' 테마를 설정한다. 테마설정
textAlign string 'left' 텍스트 정렬을 설정한다.
[ 'left', 'right' ]중 택일
readOnly boolean false 읽기전용으로 사용할지 여부를 설정한다.
promptChar string '_' (under bar) 입력 가능한 빈칸에 나타날 프롬프트 문자를 설정한다.

xgInitOption

Name Type Default Description
xgDataSet object|string null 데이터 셋을 설정한다. 데이터 셋 오브젝트 또는 데이터 셋 아이디를 지정할 수 있다.
xgDataTable object|string null 데이터 테이블을 설정한다. 데이터 테이블 오브젝트 또는 데이터 테이블 아이디를 지정할 수 있다.
해당 데이터 테이블은 같이 지정된 데이터 셋에 추가된 상태여야 한다.
xgBindOption

Name Type Default Description
mode string 'read' 데이터 바인딩 모드를 설정한다. 'bind.type' 이 'row' 일 경우에는 'read' 모드만 지원된다.
[ 'read', 'write' ] 중 택일
  • 'read' : 해당 컴퍼넌트의 변경사항이 타 컴퍼넌트에게 전달되지 않는다.
  • 'write' : 해당 컴퍼넌트의 변경사항이 타 컴퍼넌트에게 전달된다.
immediately boolean true 'write' 모드 일 때 변경사항을 즉시 전파할지 여부를 설정한다. false로 설정할 경우 컴퍼넌트에서 포커스를 잃을 때 변경사항을 전파다.
bind object {} 오브젝트 구조는 여기를 참조한다.
bind object

Name Type Default Description
type string 'column' 데이터 테이블의 바인딩 목표를 설정한다. 데이터 테이블의 현재 선택된 로우를 기준으로 동작된다.
'column' 모드만 지원된다.
column string|number null 'column' type으로 설정되었을 때 바인딩 할 컬럼을 지정한다. 컬럼의 이름 또는 컬럼의 인덱스로 지정할 수 있다. ( 데이터 테이블 기준 )
Listens to Events:
  • valuechanged - 실질적인 값이 바뀌었을 때 발생한다 ( 실제 값과 텍스트가 다를 수 있다 )
  • change - 값이 바뀌었고 마스크 입력 컨트롤이 포커스를 잃었을 때 발생한다
  • textchanged - 텍스트가 바뀌었을 때 발생한다
Returns:

$xgMaskedInput

Type
$object
Examples
var maskedInputOption = {
    xgInitOption: {
        xgDataSet: dataSet,
        xgDataTable: 'dataTableName'
    },
    xgBindOption: {
        mode: 'read',
        bind: {
            type: 'column',
            column: 'columnName'
        }
    },
    width: 450,
    height: 30,
    ...
};

// module 방식
var $xgMaskedInputModule = new XgUIMaskedInput( '#xg-masked-input', maskedInputOption );

// plugin 방식
var $xgMaskedInputPlugin = $( '#xg-masked-input' ).xgSlider( maskedInputOption );
<!-- data-api 방식 ( Markup Tag로 선언한다. ) -->

         "xgInitOption": {

             "xgDataTable": "dataTableName"

             "mode": "read",
             "bind": {
                 "type": "column",
                 "column": "columnName"
             }
         },
         "width": 300,
         "height": 300,
         ...
         "mask": "\#\#\#-\#\#\#\#",
         "promptChar": "$"
     }'
></div>
