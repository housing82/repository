XgUIExcelButton

XgUIExcelButton Class

new XgUIExcelButton(selector, option) → {object}

새로운 엑셀 버튼 인스턴스를 생성한다. XgUIButton 컴퍼넌트를 사용함으로 공통 버튼 옵션을 동일하게 사용할 수 있다.

Parameters:
Name Type Argument Description
selector string

CSS 셀렉터 ( 하기 지정된 테그를 사용하여야 한다 )

  • 'import' : input[type="file"]
  • 'export' : button 또는 input[type="button"]
option object <optional>

엑셀 버튼 옵션
Options:
Name Type Default Description
xgInitOption object {} 데이터 셋 및 데이터 테이블 등 초기화에 필요한 내용을 설정한다. 자세한 정보는 여기를 참조한다.
xgBindOption object {} 엑셀 가져오기/내보내기 동작시 수행할 콜백함수를 설정한다. 자세한 정보는 여기를 참조한다.

xgInitOption

Name Type Default Description
xgDataSet object|string null 데이터 셋을 설정한다. 데이터 셋 오브젝트 또는 데이터 셋 아이디를 지정할 수 있다.
xgDataTable object|string null 데이터 테이블을 설정한다. 데이터 테이블 오브젝트 또는 데이터 테이블 아이디를 지정할 수 있다.
해당 데이터 테이블은 같이 지정된 데이터 셋에 추가된 상태여야 한다.
type string '' 엑셀 버튼 종류를 설정한다.
[ 'import', 'export' ] 중 택일
  • 'import' : 엑셀 가져오기
  • 'export' : 엑셀 내보내기
text string '' 엑셀 버튼에 표시될 문구를 설정한다.
'import'일 경우, 기본값은 '엑셀 가져오기', 'export'일 경우, 기본값은 '엑셀 내보내기'로 설정되어 있다.
supportOlderBrowser boolean true IE9 이하의 브라우저 지원기능 활성화 여부를 설정한다. 활성화 시에 설정된 actionUrl로 파일을 전송하며 서버에서 해당 처리가 구현되어 있어야 한다.
unsupportBrowser function function supportOlderBrowser 옵션이 꺼져있고 IE9 이하의 브라우저를 사용할 때 수행할 콜백함수를 설정한다.
xgUIWorkerUrl string './xg-platform' 브라우저에서 Web Worker를 지원할 때 사용할 XgUIWorker의 경로를 지정한다. 브라우저에서 Web Worker를 지원하지 않거나 상대경로가 기본값과 일치한다면 생략 가능하다.

브라우저별 지원여부
xlsxUrl string './lib/js-xlsx/dist' XgUIWorker에서 사용할 js-xlsx 라이브러리의 경로를 설정한다. 브라우저에서 Web Worker를 지원하지 않거나 상대경로가 기본값과 일치한다면 생략 가능하다.

위치는 'XgUIWorker.js' 파일 위치에서의 상대경로 이다.
fileName string 'xg-excel-export' 엑샐 내보내기로 만들어진 엑셀파일의 이름을 설정한다. ( 확장자는 xlsx 로 자동생성된다 )
actionUrl string 상세 설명 참조 IE9 이하 브라우저를 통한 엑셀 가져오기 / 내보내기를 수행할 때 데이터 변환을 수행할 서버 주소를 설정한다. 엑셀버튼의 타입별 기본값은 아래와 같다.

type이 'import' 일 때 - '/XG/ix-import.xlsx'
type이 'export' 일 때 - '/XG/ix-export.xlsx'

xgBindOption

Name Type Default Description
start function null 엑셀 가져오기/내보내기 기능이 시작될 때 수행할 콜백함수를 설정한다.
end function null 엑셀 가져오기/내보내기 기능이 종료될 때 수행할 콜백함수를 설정한다.
See:
  • 버튼 옵션
Returns:

$xgExcelButton

Type
object
Examples
var importOption ={
    xgInitOption: {
        type: 'import',
        xgDataSet: dataSet,
        xgDataTable: 'dataTableName'
    },
    xgBindOption: {
        start: function () {
            alert('excel import start');
        },
        end: function () {
            alert('excel import end');
        }
    }
};

// module 방식
var $xgExcelButtonModule = new XgUIExcelButton('#xg-excel-button', importOption);

var exportOption ={
    xgInitOption: {
        type: 'export',
        xgDataSet: dataSet,
        xgDataTable: 'dataTableName'
    }
};

// plugin 방식
var $xgExcelButtonPlugin = $('#xg-excel-button').xgExcelButton(exportOption);
<!-- data-api 방식 (Markup Tag로 선언한다.) -->
<!-- 엑셀 가져오기 -->
<input
    type="file"
    data-xg-excel-button
    data-xg-option='{
        "template": "success",
        "xgInitOption": {
            "type": "import",
            "xgDataSet": "dataSetName",
            "xgDataTable": "dataTableName",
            "xgUIWorkerUrl": "./xg-platform",
            "xlsxUrl": "lib/dist/js-xlsx"
        }
    }'
>

<!-- 엑셀 내보내기 -->
<button
    type="button"
    data-xg-excel-button
    data-xg-option='{
        "template": "danger",
        "xgInitOption": {
            "type": "export",
            "xgDataSet": "dataSetName",
            "xgDataTable": "dataTableName",
            "xgUIWorkerUrl": "./xg-platform",
            "xlsxUrl": "lib/dist/js-xlsx"
        }
    }'
></button>
