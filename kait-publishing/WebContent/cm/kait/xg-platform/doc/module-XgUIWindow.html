XgUIWindow

다이얼로그/윈도우를 표현하는 기능을 제공하는 컴포넌트이다.

Dependencies

Name Description Remark
jquery DOM Element를 쉽게 관리하는 기능을 제공한다. 1.11.x 이상
XgUICommon Xg UI Components의 공통 상수 및 메서드를 제공한다.

Classes

XgUIWindow

Methods

xgWindow('closeAll')

열려있는 모든 윈도우(모달이 아닌)를 닫는다.

Parameters:
Name Type Description
'closeAll' string

메서드 이름
Example
$xgWindow.xgWindow('closeAll');

윈도우의 타이틀을 설정한다.

메서드 이름

타이틀로 설정할 문자열

xgWindow('isOpen') → {boolean}

윈도우가 열려있는지 닫혀있는지 여부를 가져온다. true이면 해당 윈도우가 열려있는 상태이고, false이면 닫혀있는 상태이다.

Parameters:
Name Type Description
'isOpen' string

메서드 이름
Returns:

openStatus

Type
boolean
Example
var openStatus = $xgWindow.xgWindow('isOpen');

현재 윈도우를 지정한 크기로 조정한다. 단, 'resizable' 속성이 true로 설정되어 있어야만 동작한다.
