Name Type Default Description

column object
Name Type Default Description

sortable boolean false 컬럼의 정렬 기능 적용여부를 설정한다.
filterable boolean false 컬럼의 필터 기능 적용여부를 설정한다.

오브젝트일 경우

컬럼에 타입을 사용자가 정의하여 사용할 때 작성한다. ['datetimeinput', 'numberinput', 'checkbox', 'passwordinput', 'dropdownlist'] 를 지원한다.

datafield(string), value(object) 형식으로 작성하며 value 하기 옵션 적용 예제를 참조한다. (각 컬럼에 추가되는 옵션은 해당 컴퍼넌트의 xgInitOption 및 xgBindOption을 제외한 모든 옵션을 적용할 수 있다.)
Listens to Events:
  • rowclick - 로우가 클릭될 때 발생한다
  • rowdoubleclick - 로우가 더블클릭될 때 발생한다
  • rowselect - 로우가 선택될 때 발생한다
  • rowunselect - 로우가 선택해제될 때 발생한다
  • rowexpand - 로우 상세설명이 펼쳐질 때 발생한다
  • rowcollapse - 로우 상세설명이 접힐 때 발생한다
  • groupexpand - 그룹된 정보가 펼쳐질 때 발생한다
  • groupcollapse - 그룹된 정보가 접힐 때 발생한다
  • sort - 정렬이 일어날 때 발생한다
  • filter - 필터링이 일어날 때 발생한다
  • columnresized - 컬럼의 사이즈가 변경될 때 발생한다
  • columnreordered - 컬럼의 순서가 변경될 때 발생한다
  • columnclick - 컬럼을 클릭했을 때 발생한다
  • cellclick - 셀을 클릭했을 때 발생한다
  • celldoubleclick - 셀을 더블클릭했을 때 발생한다
  • cellselect - 셀이 선택될 때 발생한다
  • cellunselect - 셀이 선택해제될 때 발생한다
  • cellvaluechanged - 셀의 값이 변경될 때 발생한다
  • cellbeginedit - 셀의 값이 변경이 일어나는 순간 발생한다
  • cellendedit - 셀의 값이 변경이 끝나는 순간 발생한다
  • pagechanged - 페이지가 변경될 때 발생한다
  • pagesizechanged - 페이지 사이즈가 변경될 때 발생한다
  • bindingcomplete - 그리드에 데이터가 로드가 완료될 때 발생한다
  • groupschanged - 그룹의 정보가 변경될 때 발생한다
Returns:

$xgGrid

Type
object
Examples
// dataSet안에 'dataTable'이라는 데이터 테이블이 들어와있고 서버에서 자료를 받아왔다고 가정한다.
var gridOption = {

    // xgGrid 기본 옵션
    width: 730,
    height: 340,
    sortable: true
    editable: true
    editmode: 'dblclick',
    keyboardnavigation: true,
    columnsmenu: true,
    columnsresize: true,
    columnsreorder: true,
    groupable: true,
    pageable: true,
    enablekeyboarddelete: true,
    filterable: true,
    ...
    // xgGrid init option
    xgInitOption: {
        xgDataSet: dataSet,
        xgDataTable: 'dataTableName',
        columns: [
            {
                datafield: 'foo',
                text: 'foo text',
                ...
                align: 'left',
                cellsalign: 'center'
            },
            ...
            {
                datafield: 'miz',
                text: 'miz header',
                align: 'right',
                cellsalign: 'left'
            }
    },
    // xgGrid data bind option
    xgBindOption: {
        template: 'all',
        ...
        showRowStatus: fasle,
        cellValidation: false,
        useEditor: {
            'foo': { columntype: 'datetimeinput' },
            'bar': { columntype: 'passwordinput' },
            'roo': {
                columntype: 'numberinput',
                option: {
                    decimalDigits: 0,
                    max: XgCommon.constant.RANGE_OF_GAUCE.INT.MAX,
                    min: XgCommon.constant.RANGE_OF_GAUCE.INT.MIN
                }
            },
            'kal': {
                columntype: 'checkbox',
                checkedAll: true
            },
            'miz': {
                columntype: 'dropdownlist'
            }
        }
    }
};

// module 방식
var $xgGridModule = new XgUIGrid('#xg-grid', gridOption);

// plugin 방식
var $xgGridPlugin = $('#xg-grid').xgGrid(gridOption);
