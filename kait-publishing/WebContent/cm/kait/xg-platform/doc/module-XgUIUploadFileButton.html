XgUIUploadFileButton

업로드할 파일을 선택하여, 데이터테이블에 파일의 정보(파일이름, 파일크기 등등)를 추가하는 기능을 제공한다.

Dependencies

Name Description Remark
jquery DOM Element를 쉽게 관리하는 기능을 제공한다. 1.11.x 이상
json3 JSON <=> 문자열 의 변환 기능을 제공한다.
XgCommon Xg Platform의 공통 상수 및 메서드를 제공한다.
XgUICommon Xg UI Components의 공통 상수 및 메서드를 제공한다.
XgUIButton 기본 버튼 기능 외에 토글, 스위치, 드랍다운, 링크 등의 추가 기능을 제공한다

Classes

XgUIUploadFileButton

Methods

xgUploadFileButton(methodName, [, args])

공통 메서드를 실행한다. xgButton 메서드에서 사용하는 메서드는 공통으로 사용 가능하다.

Parameters:
Name Type Argument Description
methodName string

공통 메서드 이름
[, args] unknown <optional>

메서드 전달 인자
See:
  • 버튼 공통 메서드
Example
// 버튼 공통 메서드 'focus'
$xgUploadFileButton.xgUploadFileButton('focus');

// 버튼 공통 메서드 'val'
var buttonVal = $xgUploadFileButton.xgUploadFileButton('val');
