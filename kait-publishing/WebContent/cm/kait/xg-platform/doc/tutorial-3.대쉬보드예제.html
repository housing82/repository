3.대쉬보드예제(30분)

본 문서는 Xg Platform을 시작하기 ( 5분 ) 내용을 실습해본 사용자들을 대상으로 작성되었다.

준비 사항

사용자의 하기의 폴더 구조 및 서버측에 API 가 준비되어 있다고 가정한다. 만약 서버 데모 페키지를 이용하는 사용자는 로컬 주소로 변경하여 사용한다.

본 예제는 본인이 원하는 웹 서버 또는 웹 어플리케이션 서버 위에서 구동하도록 한다.

  • Xg Platform 다운로드 링크를 통해 최신버전을 다운로드를 받는다.
  • API URL
    • 정보 조회 : /XG/jsp/chartSelect.jsp
    • 정보 변경 : /XG/jsp/chartUpdate.jsp
    --- js
     +- lib
     +- style
     +- xg-platform +- doc
     |              +- examples
     |              +- lib
     index.html

작성할 어플리케이션

Xg Platform의 모듈들은 헤더에 스크립트를 포함하는 일반방식과 require.js 를 이용하여 모듈화 및 lazy loading를 이용할 수 있는 AMD 방식을 지원한다. 본 예제는 require.js 를 이용하여 모듈을 AMD 방식으로 이용하는 방법을 사용한다.

사용자의 요구 사항은 다음과 같다.

  1. 데이터 테이블의 정보를 그리드 및 차트로 표현되어야 한다.
  2. 그리드의 로우 선택 시 해당 월의 정보( 예치금, 세금 )를 바차트로 표현되어야 한다.
  3. 그리드의 컬럼 별로 파이 및 도넛 차트로 표현되어야 한다.
  4. 그리드의 모든 로우의 정보를 라인차트로 표현되어야 한다.
  5. 그리드의 셀의 값 변경 시 각각 표현되는 차트에도 반영되어야 한다.
  6. 데이터를 엑셀로 내보내기 및 가져오기 기능이 제공되어야 한다.

2. 엘리먼트 및 스크립트( js/dashboard.js ) 코드 추가

컨탠츠를 2개의 section tag로 나누고 각각의 상단 영역은 그리드와 컬럼차트, 파이차트( id="top" )를 하단 영역은 도넛차트, 라인차트( id="bottom" )을 표현 할 것이다.

4. 스크립트 코드 작성 ( js/dashboard.js )

require.config( {

설정이 끝나면 'dashboard' module을 정의한다. 최초에 서버의 데이터를 조회하기 위한 의존성 모듈만 로드한다. 의존성 모듈 로드 후 데이터 셋, 데이터 테이블, 데이터 어댑터를 생성 및 설정한다.

        ..( 생략 )..

        'XLSX': 'lib/js-xlsx/dist/xlsx.full.min',
        'FileSaver': 'lib/file-saver/FileSaver'
    }
} );

define( 'dashboard', [ 
        'jquery', 
        'XgDataAdapter', 
        'XgDataTable', 
        'XgDataSet' 
    ], function ( $, XgDataAdapter, XgDataTable, XgDataSet ) {

        // 이후에 차트설정에 사용할 숫자에 3자리 마다 콤마를 넣어 변환해줄 함수
        var numberWithCommas = function ( num ) {

            if ( isNaN( num ) ) {
                return num;
            }

            return num.toString().replace( /\B(?=(\d{3})+(?!\d))/g, ',' );
        };

        var bankDataSet = new XgDataSet( 'bankDataSet' );

        // 데이터 어댑터를 선언과 동시에 등록한다.
        bankDataSet.setDataAdapter( new XgDataAdapter().setDataType( 'JSON' ).setSOCD( '(I:chart=bankDataTable,O:chart=bankDataTable)' ) );

        // 데이터 테이블을 선언과 동시에 등록한다.
        bankDataSet.addDataTable( new XgDataTable( 'bankDataTable', {
            url: {
                select: '/XG/jsp/chartSelect.jsp',
                update: '/XG/jsp/chartUpdate.jsp'
            }    
        } ) );

        // 로드와 동시에 최초 1회 서버측에 데이터를 요청한다.
        bankDataSet.select( 'bankDataTable', function () {

        } );

} );

이제 'bankDataSet'에 등록된 'bankDataTable'을 서버와 동기화를 진행하고, 이후 콜백 실행 시 각 UI 컴퍼넌트 모듈들을 로드하고 초기화 작업을 수행한다.

콜백 함수에서는 각종 버튼 들 및 그리드, 차트들의 기능을 적용할 것이다. 작성 시 UI 컴퍼넌트들을 모듈 방식 및 플러그인 방식으로 혼용해서 사용할 것이다. ( AMD로 로드하더라도 jquery의 $.fn 함수의 확장으로 등록되어 플러그인 방식으로 사용이 가능하다는 것을 확인하기 위함이다. )

우선 조회, 저장, 엑셀 가져오기, 엑셀 가져오기 버튼을 모듈 방식으로 적용한다. 여기서 주의할 점은 엑셀 버튼의 경우 모던 브라우저에서 제공하는 Web Worker 기능을 이용하는 XgUIWorker를 이용할 경우 경로 지정 부분이다.

'xgUIWorkerUrl' 속성 값은 'xg-ui-worker.js' 상대경로를 기입하여야 하며, 'xlsxUrl' 속성 값은 'xg-ui-worker.js' 위치를 기준으로 'xlsx.full.min.js' 파일위치의 상대 경로를 지정해야한다.

다음으로 그리드를 설정한다. 이번에는 모듈 방식이 아닌 플러그인 방식으로 사용할 것이다. 그리고 초기화 시 'xgBindOption' 을 사용하지 않고 직접 메서드를 사용하여 그리드의 언어셋을 설정한다. ( 단 'xgBindOption'을 사용할 경우 브라우저의 국가코드를 읽어 자동으로 언어셋을 적용시킨다 )

그리드에는 하기 옵션을 적용시킨다. 이 외의 다양햔 옵션은 여기를 참조한다.

다음으로 차트들을 적용시킨다. 각 차트들의 용도는 다음과 같으며, 데이터가 변경되면 차트도 갱신된다.

  • column chart( '.row-column-chart' ) : 데이터 테이블에 선택된 로우의 수치화 가능한 컬럼을 막대 형태의 차트로 표현되며, 선택된 로우가 변경될 때 마다 차트가 갱신된다.
  • pie chart( '.column-pie-chart' ) : 예치금( 'DEPOSIT' ) 컬럼들의 값을 기준으로 원 형태의 차트로 표현된다.
  • donut chart( '.column-donut-chart' ) : 세금( 'TAX' ) 컬럼들의 값을 기준으로 도넛 형태의 차트로 표현된다.
  • line chart( '.full-line-chart' ) : 모든 로우의 정보를 기준으로 선 형태의 차트로 표현된다. 예치금( 'DEPOSIT' ) 및 세금( 'TAX' ) 컬럼의 정보를 중첩하여 표현한다.

                            xgDataTable: 'bankDataTable'

                            visible: false

                            minValue: -250000,
                            maxValue: 2500000,

                                return numberWithCommas( value ) + ' 원';
                            }
                        },
                        toolTipFormatFunction: function( value, itemIndex, serie ) {
                            // 하기와 같이 모듈형식으로 사용하더라도 $object로 반환되기 때문에 이후 플러그인 사용방식으로도 사용 가능하다.

                        },
                        seriesGroups: [
                            {
                                type: 'column',
                                series: [
                                    {

                                        displayText: '세금'
                                    }
                                ]
                            }
                        ]
                    } );

                        xgInitOption: {
                            xgDataSet: 'bankDataSet',
                            xgDataTable: 'bankDataTable'
                        },
                        xgBindOption: {
                            bind: {
                                type: 'column',
                                column: 'DEPOSIT'
                            }

                        description: '2014년 기준(pie chart)',

                        showLegend: false,
                        showToolTips: false,
                        seriesGroups: [

                                showLabels: true,

                                    }

                                        dataField: 'DEPOSIT',
                                        displayText: '예치금'

                            bind: {
                                type: 'column',
                                column: 'TAX'
                            }
                        },
                        title: '월별 세금 현황',
                        description: '2014년 기준(donut chart)',
                        width: '100%',
                        height: 300,
                        showLegend: true,
                        seriesGroups: [

                                type: 'donut',

                            xgDataSet: 'bankDataSet',

                        xgBindOption: {
                            bind: {
                                type: 'column',
                                column: 'DEPOSIT'
                            }

                        showLegend: true,

                                    month = '0' + month;
                                }
                                return date.getFullYear() + '-' + month;
