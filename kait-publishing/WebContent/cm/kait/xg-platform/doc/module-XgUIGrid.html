XgUIGrid

XgCommon Xg Platform의 공통 상수 및 메서드를 제공한다.
XgUICommon Xg UI Components의 공통 상수 및 메서드를 제공한다.

xgGrid('beginupdate', suspendAll)

한번에 메서드 또는 속성설정을 다중 수행할 경우 묶어서 실행한다. 2번째 인자로 true를 전달하게 되면 'endupdate' 메서드가 실행되기 전 까지의 모든 랜더링이 수행되지 않는다.

Parameters:
Name Type Argument Description
'beginupdate' string

메서드 이름
suspendAll boolean <optional>

랜더링 연기 여부
Example
$xgGrid.xgGrid( 'beginupdate', true );

$xgGrid.xgGrid( 'setcolumnproperty', 'foo', 'width', 100 );
...
$xgGrid.xgGrid( 'setcolumnproperty', 'bar', 'align', 'right' );

$xgGrid.xgGrid( 'endupdate' );

xgGrid('addfilter', datafield, filterGroup, apply, customfilter)

그리드에 사용자 정의 필터를 추가한다.

Parameters:
Name Type Argument Description
'addfilter' string

메서드 이름
datafield string

데이터 필드 값
filterGroup object

다중 필터가 추가된 필터 그룹
apply boolean <optional>

즉시 필터 그룹을 적용할지 여부
customfilter function <optional>

정의된 필터를 사용하지 않을 경우 수행할 사용자 정의 필터 함수
전달 인자 : filterValue, cellValue, condition
See:
  • 필터 타입 및 조건
Example
// 필터를 추가할 필터 그룹 인스턴스를 생성한다.
var filterGroup = new $.xgfilter();
var numericCompareValue = 10;
var stringCompareValue = 'bar';

// 숫자형( 비교 값 이하만 ) 및 문자형( 비교 값으로 시작하고 대소문자를 구분 ) 필터를 생성한다.
var numericFilter = filterGroup.createfilter( 'numericfilter', numericCompareValue, 'LESS_THAN_OR_EQUAL' );
var stringFilter = filterGroup.createfilter( 'stringfilter', stringCompareValue, 'STARTS_WITH_CASE_SENSITIVE' );

// 필터 그룹에 필터를 추가한다. 첫번째 인자인 flag는 다음과 같다. 0 : 'and'의 의미, 1 : 'or'의 의미
var flag = 0;
fiilterGroup.addfilter( flag, numericFilter );
fiilterGroup.addfilter( flag, stringFilter );

// 필터 그룹을 그리드에 추가한다.
$xgGrid.xgGrid( 'addfilter', 'foo', fiilterGroup );

// 또는 필터 그룹을 추가와 동시에 적용한다.
$xgGrid.xgGrid( 'addfilter', 'foo', fiilterGroup, true );

그리드에 로우 데이터를 추가한다. addPosition을 지정하지 않으면 마지막에 추가된다.

Parameters:
Name Type Argument Description
'addrow' string

메서드 이름
rowId number

로우 아이디
데이터 셋과 연동 시 내부에서 아이디를 자동으로 생성하여 관리함으로 반드시 null로 전달한다.
rowData object | array

로우 데이터 오브젝트 또는 배열
addPosition string <optional>

새로운 로우가 추가될 위치
[ 'first', 'last' ] 중 택일
Returns:

$xgGrid

Type
$object
Example
// 빈 로우를 추가한다
$xgGrid.xgGrid( 'addrow', null, {} );

// rowData를 Object로 전달. 첫번째 로우로 추가된다.
$( '#xg-grid' ).xgGrid( 'addrow', null, { foo: 20, bar: 'david' }, 'first' );

// rowData를 Array로 전달
$xgGrid.xgGrid( 'addrow', null, [ { foo: 18, bar: 'chase' }, { foo: 40, bar: 'rona' } ] );

xgGrid('removerow', rowIdx) → {$object}

로우 인덱스에 해당하는 그리드 로우를 삭제한다. 다중 로우를 삭제하려면, 로우 아이디 배열을 전달한다.
데이터 셋과 연결되어 있을 시 실제 로우가 삭제되지 않고 로우 상태가 삭제 상태로 변경된다.

Parameters:
Name Type Description
'removerow' string

메서드 이름
rowIdx number | array

로우 인덱스 또는 로우 인덱스 배열
Returns:

$xgGrid

Type
$object
Example
// 단일 삭제
$xgGrid.xgGrid( 'removerow', 0 );

// 다중 삭제
var rowIds = [];
var rows = $xgGrid.xgGrid( 'getboundrows' );

for( var idx in rows ) {
    rowIds.push( rows[ idx ].uid );
}

$xgGrid.xgGrid( 'removerow', rowIds );
