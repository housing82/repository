<static> deepFreeze(obj) → {object}

오브젝트를 수정불가능 한 상태로 변경한다. 내부 중첩 오브젝트에 대해서도 freeze 속성을 적용시킨다.
IE9 이하에서는 동작하지 않는다

Parameters:
Name Type Description
obj object

오브젝트
Returns:

frozenObject

Type
object
Example
XgCommon.deepFreeze( obj );

<static> deleteProperties(object, properties) → {object}

object에서 특정 속성을 제거한다. 2번째 인자가 문자열일 경우 해당 속성만 제거하고 배열로 전달될 경우 다중 속성을 제거한다.

Parameters:
Name Type Description
object object

속성을 지울 대상 오브젝트
properties string | array

삭제 할 속성명 또는 속성명을 지닌 배열
Returns:

XgCommon

Type
object
Example
var object = {
   foo: 1234,
   bar: false
};
XgCommon.deleteProperties( object, 'foo' );

XgCommon.deleteProperties( object, [ 'foo', 'bar' ] );
