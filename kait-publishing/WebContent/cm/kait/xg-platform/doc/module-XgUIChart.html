XgUIChart

다양한 데이터를 차트형식으로 시각화하는 기능을 제공한다.

REMARK : 현재 데이터 바인딩은 'column', 'line', 'pie' 차트에서만 제공된다

Dependencies

Name Description Remark
jquery DOM Element를 쉽게 관리하는 기능을 제공한다. 1.11.x 이상
XgCommon Xg Platform의 공통 상수 및 메서드를 제공한다.
XgUICommon Xg UI Components의 공통 상수 및 메서드를 제공한다.

xgChart('saveAsJPEG', fileName, serverUrl)

차트를 JPEG로 변환하여 저장한다. 브라우저는 캔버스를 지원해야하며, 장치는 네트워크에 연결되어 있어야 하여 base64로 인코딩 되어 전송된다.

Parameters:
Name Type Description
'saveAsJPEG' string

메서드 이름
fileName string

저장할 파일 이름
serverUrl string

요청을 전송할 서버 주소
Example
$xgChart.xgChart( 'saveAsJPEG', 'http://myserver.com/url' );
